# Installation
step-1. Clone the repository
 - git clone https://github.com/zinzala/flutter-tic-tac-toe.git

step-2. Navigate to the project directory
 - tic_tac_toe
  
step-3. Install dependencies
 - flutter pub get

step-4. Run the app
 - flutter run
